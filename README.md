## Development

```bash
yarn install
yarn dev-server
```

### Build

```bash
yarn install
yarn build
```

Output is available under `build/` directory.

## Deployment

Every push to `master` branch is automatically deployed to https://asyncapi-jsonschema.com/ using a Google Cloud Build trigger.
